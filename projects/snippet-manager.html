Code Snippet Manager

Status: Active
Date: November 2025
Role: Creator & Maintainer

About This Project

A powerful productivity tool designed specifically for developers to organize, search, and share code snippets efficiently. The application supports syntax highlighting for over 50 programming languages and provides a clean, intuitive interface for managing code collections.

Built with TypeScript and Node.js, this tool emphasizes speed and reliability. It features a robust search system with tag-based filtering, favorites management, and the ability to export snippets in various formats. The project is open-source and actively maintained.

What started as a personal productivity hack has grown into a tool used by hundreds of developers. The project taught me valuable lessons about building developer tools, API design, and creating software that truly solves real problems.

Technologies Used

TypeScript Node.js Express MongoDB React Prism.js

Key Features

  • Support for 50+ programming languages with syntax highlighting
  • Fast full-text search with tag-based filtering
  • Organize snippets into collections and categories
  • Export snippets to Markdown, JSON, or plain text
  • Share snippets with unique URLs and access controls
  • Dark mode and light mode themes
  • Keyboard shortcuts for power users
  • Browser extension for quick snippet access
